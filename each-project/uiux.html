Project Title: STREETSTYLE Clothing Site

Total Hours: 25 Hours

VIEW PROJECT

The goal of this project is to create a high-fidelity e-commerce website using UI/UX techniques learned in class with Figma for mobile devices. With STREETSTYLE users can walk through the website to browse and purchase high quality streetwear.

Brainstorming Process:

The project began with brainstorming the outline and creating a style guide. This was followed by developing the user flow and personas to gain a clear understanding of the website’s overall experience. Studying the target audience and their specific needs laid the foundation for how the website would look and operate. The brainstorm sessions involved using Figma and various design techniques to develop scalable sketches and prototypes. With the help of Figma, the brainstorming process was highly efficient and provided a clear vision for what this project would mean to me.

Website Concept and Rough Drafts:

STREETSTYLE's concept drew inspiration from multiple high-end fashion retailers, mostly from streetwear style and mobile-friendly websites. The concept sketches and wireframes of the high-fidelity rough draft and the style guide were also created in Figma.

Information Gathering and Image Collection:

The text was written with the help of ChatGPT after conducting research on streetwear and other e-commerce websites. Images were collected by comparing existing websites with the target concept and target audience, following the latest trends. The website design was influenced by popular e-commerce sites such as StockX and Off-White.

Website Creation:

The website creation process went through multiple design phases, starting with Figma for brainstorming and prototyping. The final website was created in code with close attention paid to the mobile-friendly design and user experience. Prototype testing was essential to maintaining an organized layout and consistent spacing. The prototype feature was used to test how the website works. It helped identify any problems and make the website easier to use before it was completed.
